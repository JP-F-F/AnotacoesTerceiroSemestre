# TP_10 - Padrões Criacionais Prototype

O prototype é um tipo de design que ajuda a criar cópias de objetos sem precisar mudar p código de suas classes originais. Permitindo criar objetos a partir de padrões já existentes.

Para fazer isso nós criamos uma classe protótipo que possui o método para clonagem.

Isso é útil quando você precisa criar muitos objetos parecidos entre si, pois economiza tempo e esforço de programação.

Exemplo de aplicação:
Criação de documentos. 
* Primeiro, cria-se uma classe "Documento" que contém os campos básicos comuns a todos os tipos de documentos.
* Em seguida, cria-se uma subclasses específicas para cada tipo de documento, como "Relatório", "Carta" e "Contrato", que adicionam campos específicos os respectivos tipos de documento.    

## EXEMPLO DE CÓDIGO

````javascript

// Classe que será clonada
class Pessoa{
    constructor(id, nome, idade){
        this.id = id;
        this.nome = nome;
        this.idade = idade;
    }

    //Método de clonagem que cria uma cópia superficial do objeto
    clone(){
        return new Pessoa(this.id, this.nome, this.idade)
    }
}

// Classe pessoa gerente para gerenciar estâncias de pessoa
class PessoaGerente{
    constructor(){
        this.pessoas = {};
    }

    // Adiciona uma nova pessoa ao dicionário de pessoas
    adicionarPessoa(id, nome, idade){
        const pessoa = new Pessoa(id, nome, idade);
        this.pessoas[id] = pessoa;
    }

    // Solicita uma pessoa pelo id e retorna uma cópia dela
    getPessoaById(id){
        const pessoaOriginal = this.pessoas[id];
        if(pessoaOriginal){
            return pessoaOriginal.clone();
        } else{
            return null;
        }
    }
}

// Criando uma instância de PessoaManager
const manager = new PessoaManager();

// Adicionando duas pessoas
manager.adicionarPessoa(1, 'joao', 30);
manager.adicionarPessoa(2, 'Maria', 25);

// Clonando a primeira pessoa e modificando o nome da cópia
const pessoaClone = manager.getPessoaById(1);
if(pessoaClone){
    pessoaClone.nome = 'joao clonado';
}

// Imprimindo as duas pessoas
console.log('Pessoa Original:');
console.log(manager.getPessoaById(1));
console.log('Pessoa Clonada:');
console.log(pessoaClone);
````
