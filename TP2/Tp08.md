# Tp08 - Padrões Criacionais Abstract factory

O padrão de projeto Abstract Factory é uma forma de criar conjuntos de objetos que estão relacionados sem precisar dizer exatamente qual é a classe de cada um deles. Ao invés de criar um objeto específico, a fábrica cria famílias inteiras de objetos que funcionam juntos.

O padrão de projeto Abstract Factory é útil quando o código precisa trabalhar com várias famílias de produtos relacionados, mas não quer depender de classes concretas desses produtos, permitindo futura escalabilidade.
Ele fornece uma interface para a criação de objetos de cada classe das famílias de produtos, prevenindo a criação de uma variante errada de um produto que não coincide com produtos já criados pela aplicação.
É recomendado implementar o Abstract Factory quando uma classe lida com múltiplos tipos de produto, para manter a responsabilidade de cada classe bem definida.

O padrão de projeto Abstract Factory é _usado em aplicativos multiplataforma_ para criar elementos de interface gráfica de usuário que se parecem diferentes em diferentes sistemas operacionais, mas que funcionam de forma semelhante.

Para isso, a fábrica abstrata declara um conjunto de métodos de criação que são utilizados pelo código cliente para produzir diferentes tipos de elementos de UI que correspondem ao sistema operacional utilizado.

# Código de Exemplo

```JavaScript
// PRODUTOS ABSTRATOS
class ProdutoEletronico{
    detalhes(){
        throw new Error("detlahes() deve ser implementado");
    }
}

class ProdutoModa{
    detalhes(){
        throw new Error("detalhes() deve ser implementado");
    }
}

// PRODUTOS CONCRETOS - LINHA ELETRONICA
class TelefoneTech extends ProdutoEletronico{
    detalhes(){
        return "Telefone Tech: 256Gb, 5G, foco em desempenho";
    }
}

class CamisaTech extends ProdutoModa{
    detalhes(){
        return "Camisa Tech: dry-fit, estampa, foco em conforto";
    }
}

// PRODUTOS CONCRETOS - LINHA MODA
class TelefoneFashion extends ProdutoEletronico{
    detalhes(){
        return "Telefone Fashion: 128Gb, 4G, foco em beleza";
    }
}

class CamisaFashion extends ProdutoModa{
    detalhes(){
        return "Camisa Fashion: Algodão, lisa, foco em beleza";
    }
}

// FABRICA ABSTRATA
class FabricaProduto{
    criarEletronico(){
        throw new Error("criarEletronico() deve ser implementado")
    }

    criarModa(){
        throw new Error("criarModa() deve ser implementado")
    }
}

// FABRICA CONCRETA
class FabricaEletronico extends FabricaProduto{
    criarEletronico(){
        return new TelefoneTech();
    }

    criarModa(){
        return new CamisaTech();
    }
}

class FabricaModa extends FabricaProduto{
    criarEletronico(){
        return new TelefoneFashion();
    }

    criarModa(){
        return new CamisaFashion();
    }
}

// CLIENTE
function lojaVirtual(fabrica){
    const produtoEletronico = fabrica.criarEletronico();
    const produtoModa = fabrica.criarModa();

    console.log(produtoEletronico.detalhes());
    console.log(produtoModa.detalhes());
}

// USO
lojaVirtual(new FabricaEletronico());
console.log("------")
lojaVirtual(new FabricaModa());

```
