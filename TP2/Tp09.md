# Tp_09 catalogo GOF - Padrões criacionais builder

O padrão builder é usado para criar objetos complexos passo a passo, permitindo assim a criação de diferentes tipos e representações de um objeto usando o mesmo código de construção.

Assim ao invés de passarmos todos os parâmetros em um único construtor gigantesco, o builder permite que você crie o objeto aos poucos, utilizando um conjunto de objetos chamados **builder**.

Cada **builder** é reponsável por construir uma parte do objeto, assim o cliente pode combinar essas partes para criar o objeto final.

## CÓDIGO DE EXEMPLO

````javaScript

// Definindo partes de um carro

class Motor{
    construtctor(tipo){
        this.tipo = tipo;
    }
}

class Carroceria {
    constructor(estilo){
        this.estilo = estilo;
    }
}

class Rodas{
    constructor(tamanho){
        this.tamanho = tamanho;
    }
}

class Interior{
    constructor(cor){
        this.cor = cor;
    }
}

// Builder
class CarroBuilder{
    constructor(){
        this.motor = null;
        this.carroceria = null;
        this.rodas = null;
        this.interior = null;
    }

    adicionarMotor(tipo){
        this.motor = new Motor(tipo);
        return this;
    }

    adicionarCarroceria(estilo){
        this.carroceria = new Carroceria(estilo);
        return this;
    }

    adicionarRoda(tamanho){
        this.roda = new roda(tamanho);
        return this;
    }

    adicionarInterior(cor){
        this.interior = new interior(cor);
        return this;
    }

    construir(){
        return new Carro(this.motor, this.carroceria, this.rodas, this. interior);
    }
}

// Construindo o carro

class Carro{
    constructor(motor, carroceria, rodas, interior){
        this.motor = motor;
        this.carroceria = carroceria;
        this.rodas = rodas;
        this.interior = interior;
    }

    mostrarDetalhes(){
        console.log(`Carro com motor ${this.motor.tipo}, carroceria ${this.carroceria.estilo}, Rodas de tamanho ${this.rodas.tamanho} e interior na cor ${this.interior.cor}.`);
    }
}

//Usando o builder

const builder = new CarroBuilder();

const carroEsportivo = builder
    .adicionarMotor('V8')
    .adicionarCarroceria('Esportiva')
    .adicionarRodas(18)
    .adicionarCarroceria('Preto')
    .construir();

const carroSedan = builder
    .adicionarMotor('4 cilindros')
    .adicionarCarroceria('Sedan')
    .adicionarRodas(16)
    .adicionarCarroceria('Bege')
    .construir();

carroEsportivo.mostrarDetalhes();
carroSedan.mostrarDetalhes();
````
